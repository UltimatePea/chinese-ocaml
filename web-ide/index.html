骆言语言特色

骆言是一个中文编程语言，支持古雅体语法

  • 中文关键字：设、为、夫、者、受、焉等
  • 引用标识符：「变量名」
  • 字符串语法：『字符串内容』
  • 注释语法：「：注释内容：」

IDE功能

专业的Web IDE，支持骆言语言开发

  • 语法高亮：专门为骆言优化的代码着色
  • 智能搜索：快速定位文件和代码
  • 多主题：深色/浅色主题切换
  • 代码折叠：支持函数和块级折叠
