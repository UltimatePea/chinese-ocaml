## 快速开始

如果你是第一次接触骆言，建议按以下顺序阅读：

1. 首先阅读 **语言设计概述** 了解项目背景
2. 然后查看 **编程实例** 获得直观印象
3. 深入学习 **语法规范** 掌握语法细节
4. 探索 **语言特性详解** 了解高级功能
5. 最后参考 **编译器架构** 理解实现原理
